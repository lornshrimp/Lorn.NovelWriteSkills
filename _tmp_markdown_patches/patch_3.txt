-灵感萌芽
-看到轻小说分类里，多出来了一些“房东”“租客”形式的恋爱文，感觉这种身份挺有意思——确定男女主之间的另类关系——房东与租客
-一直想要尝试以高中为背景的校园题材，写一本偏日常风格的恋爱文——确定题材背景——国内高中校园
-比较经典的女主设定——傲娇的青梅竹马——擅长音乐的高冷同桌——幽默风趣的学姐（在此基础上做一些人设反差设定，看自己喜好）
-围绕灵感，构建主线框架
-主角事业线（思考路径）
-高中校园背景，不方便做生意——可以文抄——小说，歌曲，电视剧剧本
-女主的设定尽量和男主事业线挂钩——女一青梅竹马，父亲是影视公司CEO——女二擅长音乐，母亲是过气女歌手——女三也写小说，父亲是大学文学教授
-想要在主角文抄的小说里，让里面的角色文抄歌曲，再用爆款歌曲带动小说销量——魔改《我真是大明星》，主角创作《我真不是明星》，在这本文娱文中，文抄后世才会出现的爆款歌曲——通过女二搭上她母亲，卖歌进入音乐圈——拥有小说影响力后，再通过女一父亲，卖出文抄的电视剧剧本——小说、歌曲、电视剧三步走——实现财富自由
-第一阶段大纲
-目标：初步实现男女主之间的房东租客关系
-拆分需求点：主角家中要有空房，最好距离高中近
-推演：有空房的话，家庭条件太好了，最好是贷款买，背房贷有压力
-继续推演：主角设定为重生，为了展现重生金手指的效果，买房的决定应当由主角推动，所以父母一开始需要用这笔钱做一个错误的决定，再由主角逆转
-最终敲定：父母上一世选择了投资亲戚的房地产项目，结果钱被套牢，而原本可以入手的学区房，后世价值却节节攀升，这一世由主角逆转
-逆转理由：主角的改变，中考的优异成绩，考入地区最好的高中，让父母下定决心买学区房
-第一阶段结束节点：学区房到手，高中即将开学，男女主入住学区房
-确定开头切入点
-按照上述大纲，确定开头切入点应为中考前
-如果放到前一个月，时间太久，节奏不够快，毕竟本书主要内容在高中
-因此直接从中考前一天切入重生，为了方便主角逆转成绩，同时合理化文抄的记忆力，直接给主角设计记忆宫殿金手指，可以记忆上一世的东西

+### 建立并梳理细纲
+
+- 码字之前，以大纲和细纲为导向，勾勒章纲

+## 拆解示例
+
+### 灵感萌芽
+
+- 看到轻小说分类里，多出来了一些“房东”“租客”形式的恋爱文，感觉这种身份挺有意思——确定男女主之间的另类关系——房东与租客
+- 一直想要尝试以高中为背景的校园题材，写一本偏日常风格的恋爱文——确定题材背景——国内高中校园
+- 比较经典的女主设定——傲娇的青梅竹马——擅长音乐的高冷同桌——幽默风趣的学姐（在此基础上做一些人设反差设定，看自己喜好）
+
+### 围绕灵感，构建主线框架
+
+- 主角事业线（思考路径）
+- 高中校园背景，不方便做生意——可以文抄——小说，歌曲，电视剧剧本
+- 女主的设定尽量和男主事业线挂钩——女一青梅竹马，父亲是影视公司CEO——女二擅长音乐，母亲是过气女歌手——女三也写小说，父亲是大学文学教授
+- 想要在主角文抄的小说里，让里面的角色文抄歌曲，再用爆款歌曲带动小说销量——魔改《我真是大明星》，主角创作《我真不是明星》，在这本文娱文中，文抄后世才会出现的爆款歌曲——通过女二搭上她母亲，卖歌进入音乐圈——拥有小说影响力后，再通过女一父亲，卖出文抄的电视剧剧本——小说、歌曲、电视剧三步走——实现财富自由
+- 第一阶段大纲
+- 目标：初步实现男女主之间的房东租客关系
+- 拆分需求点：主角家中要有空房，最好距离高中近
+- 推演：有空房的话，家庭条件太好了，最好是贷款买，背房贷有压力
+- 继续推演：主角设定为重生，为了展现重生金手指的效果，买房的决定应当由主角推动，所以父母一开始需要用这笔钱做一个错误的决定，再由主角逆转
+- 最终敲定：父母上一世选择了投资亲戚的房地产项目，结果钱被套牢，而原本可以入手的学区房，后世价值却节节攀升，这一世由主角逆转
+- 逆转理由：主角的改变，中考的优异成绩，考入地区最好的高中，让父母下定决心买学区房
+- 第一阶段结束节点：学区房到手，高中即将开学，男女主入住学区房
+
+### 确定开头切入点
+
+- 按照上述大纲，确定开头切入点应为中考前
+- 如果放到前一个月，时间太久，节奏不够快，毕竟本书主要内容在高中
+- 因此直接从中考前一天切入重生，为了方便主角逆转成绩，同时合理化文抄的记忆力，直接给主角设计记忆宫殿金手指，可以记忆上一世的东西
+- 单纯中考逆转，情绪不太够，把青梅竹马拉进来，上一世因为青梅竹马一直被父母当做别人家的孩子，给自己上压力，导致两人在中考前关系决裂
+- 这一世直接重生到决裂当时，挽回的同时，展开女一青梅竹马的感情线，补足开头情绪
+
+### 细化大纲内容
+
+- 根据第一阶段大纲，对本阶段大纲内容进行细化构建，下方为示例
+- 开局青梅竹马决裂——重归于好——中考压力逼近
+
+### 信息差
+
+- 重生后，性格稳重，能处理好和青梅竹马之间的小情绪，不再叛逆
+
+- 对中考稍微有些把握，起码比上辈子强一点
+
+### 人际关系与情绪
+
+- 女一青梅竹马，担心主角中考却不被领情，十分伤心，结果主角重生，一番沟通，重归于好
+
+- 男配好兄弟，学霸一枚，担心主角中考
+
+- 男配副班长，暗恋女一，不看好主角
+
+- 女配，是女一的班上朋友，不好看主角，为女主打抱不平
+
+- 主角重生后的感慨，对男配女配的嘲讽并不在意，只觉得在看小孩，性格稳定，成熟，并思考中考考取高分的可行性
+
+- 回家筹备中考——引出金手指记忆宫殿（后面三个就不具体罗列了）
+
+### 信息差
+
+- 主角发现脑中的记忆宫殿，中考已经不成问题
+
+- 女主吃惊以及父母都并不知情
+
+- 人际关系和情绪
+
+- 女一依旧担心，尽心尽力为主角复习功课，争取让他哪怕多几分也好
+
+- 父母私下商议普高的择校费，担心儿子考不上普高，但也打算花钱送他去普高上学
+
+- 主角，重生后懂事了，让父母感到欣慰
+
+### 中考逆袭
+
+- 网文签约——铺垫网文事业线
+- 购买学区房
+- 建立并梳理细纲（因为大纲中罗列的事件，未必完全按照顺序一个一个发生，因此需要细纲梳理时间线顺序）
+- 以上述大纲内容示例细纲：
+- 6月13日——重生，开局切入
+- 6月14日——中考开始——亲戚登场，铺垫房地产投资
+- 6月15日——中考结束，晚饭聊起房地产投资，主角推荐买学区房，父母表示慎重考虑，主角说，学区房离重点高中近，他上学方便，但父母不信他考得上
+- 6月16日——早起去父母的早餐店里帮忙，让父母感受到自己的变化，下午则去女一家里串门，借用电脑创作网文——和女一的互动日常，交代女一家庭背景
+- 6月22日——中考成绩出炉——父母，女一，男配兄弟的震惊——父母转变态度，决定买学区房
+- 6月25日——初中毕业典礼——主角装逼
+- 7月1日——和母亲坦白网文创作，借助母亲，完成未成年签约流程
+- 7月15日——女主父亲请客吃饭——初步铺垫女一父亲的身份
+- 7月底——编辑推荐主角和女三在QQ上认识，互相章推
+- 8月初——主角带着女一去学区房入住，女三之前就租住在此，三人见面——女三问女一，主角的QQ号，结果发现和编辑推荐给自己的是同一个——而主角上辈子就认识女三，也知道对方写网文的身份
+- 8月底——高中开学

+### 码字之前，勾勒章纲
+
+- 按照细纲的时间线，以及大纲设定的事件信息差和人际关系以及情绪，梳理章纲内容
+- 示例：
+- 重生回到青梅竹马决裂当天
+- 主角中考摆烂，拉着朋友要去打球，女一想拉着他回家复习，被他一把甩开，结果女一摔倒在地，于是两人决裂
+- 主角重生后，看着女一从地上起身，哭着离开，立马追上去，大声道歉，获得原谅
+- 男配女配负责提供负面情绪，衬托女一的白给和对主角的关心
+- 回家后开始复习，主角发现记忆宫殿
+- 父母回家，看到儿子在女一帮助下复习，稍显欣慰，但在厨房做饭时还是担忧，表示要准备好择校费
+- 后面就不继续了，差不多就这个效果，章纲写的细致或者简单一点都可以，重点只在于能帮助你码字的思路保持通畅，不会卡文，所以只要有辅助的效果即可
