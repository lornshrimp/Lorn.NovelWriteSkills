# 抽象梯与事实转译补强

适用于正文里“信息是对的，但读起来不够顺、不够懂、不够有画面”的情况。

## 一、术语第一次出现要尽量落地

- 角色能懂、读者未必懂的术语，第一次出现时尽量顺手解释。
- 解释要贴在剧情里，不要写成百科脚注。

更好的做法：

- 术语 + 人话翻译
- 专业名词 + 当前场景里它到底意味着什么

## 二、事实不是越多越强，而是越有用越强

- 不要因为查到了很多资料，就都塞进正文。
- 优先保留能帮读者理解当前风险、规模、代价和异常的事实。
- 对人物、冲突、气氛没有帮助的资料，宁可删。

## 三、让数字有体感

- 单独一个大数字常常没有感觉。
- 可以把它换算成熟悉的规模、距离、人数、城市体量、价格对比。
- 目标不是炫数字，而是让读者知道“这到底有多大、多贵、多离谱”。

## 四、抽象梯：不要一直停在中间层

很多发虚的句子，都卡在既不够具体、又不够有意义的中间层。

更稳的走法：

1. 先落一个具体物或动作
2. 再让人物对它产生反应
3. 最后再上升到更大的情绪或主题感

例如：

- 先写回执、门牌、杯口、手指、灯影
- 再写角色如何看它、摸它、误判它
- 最后让读者感觉到羞耻、危险、孤独、失控或权力差

## 四点五、从细节到主题，不要跳级喊话

主题最好是“长出来”的，不是“宣布出来”的。

更稳的四步走：

1. 先给一个足够具体、可反复回看的物或动作
2. 让人物对它作出有偏向的反应
3. 让局势因为这个细节发生一点偏斜
4. 读者随后自己感觉到更大的意思

例如：

- 先有一张没撕干净的回执
- 再有人看见它时手顿了一下
- 接着对话和判断都围着它开始变味
- 最后读者自然会感到：旧事没过去，权力关系也没干净

## 四点六、主题不只一种，可以分层显影

同一段里常见三层：

- 表层：正在发生什么事
- 心理层：这个人当下感到什么、怕什么、硬撑什么
- 主题层：这件事隐隐在说更大的什么

润色时不要试图每层都说满。

通常只需要：

- 表层写实
- 心理层稍露
- 主题层点到即止

层层都喊满，最容易变说教。

## 四点七、小物件复现，是最稳的上升法之一

如果某个物件、动作、声音能在前后两处复现，它就容易从“细节”慢慢长成“主题抓手”。

可复现对象：

- 杯子 / 回执 / 门牌 / 药盒 / 雨伞 / 工牌
- 某句老话
- 某个固定动作
- 某种环境声

第一次出现：先当事实写。

第二次出现：让处境变化。

第三次出现：它常常自己就带上意义了。

## 四点八、别让人物替作者站上讲台

若一段话听起来像：

- 作者终于找到机会发表看法
- 人物突然变得比平时更会总结人生
- 台词只负责传题眼，不负责当前关系和风险

就要警惕它在“替作者说教”。

更好的做法：

- 让人物只说他在此刻真会说的话
- 让主题通过冲突、选择、代价和小物件显影
- 允许主题先模糊，后面再慢慢聚焦

## 五、比喻和类比要帮读者懂，不是帮作者显眼

- 好比喻最好接近日常经验，一看就能明白。
- 好类比优先用在：
  - 难懂概念
  - 难感知规模
  - 难进入的技术过程
- 如果比喻本身比原文还难懂，就不要留。

## 六、句长要有节拍，不要一锅粥

- 短句适合：危险、判断、反应、撞击点。
- 长句适合：延宕、压迫、困惑、复杂观察。
- 若一段全是同长度句子，读感很容易发平。

## 七、片段句与短段要慎用但别怕用

- 当需要强调、停顿、补刀或制造骤然感时，片段句是有效工具。
- 但不能满屏都在断，不然会像故意摆姿势。

## 八、围绕主题，不要写着写着跑偏

- 润色时最容易犯的错，是看见有趣信息就舍不得删，结果把章心写散。
- 每次补一段描写或事实前，先问：
  - 这段是否还在服务本章当前问题？
  - 这段是否在服务人物、冲突、主题或气氛？

如果都不是，再好看也要警惕。

## 九、细节选取的主题纪律

不是所有漂亮细节都值得留下。

优先保留的细节：

- 会改变人物判断的
- 会改变关系气氛的
- 会参与前后呼应的
- 会让本章主题更清楚但不更直白的

优先删除的细节：

- 只有作者观察力，没有叙事作用
- 跟本章问题无关
- 换成别的也不影响任何东西
