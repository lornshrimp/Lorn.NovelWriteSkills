# 去斧凿痕与透明叙述校准卡

适用于正文已经"信息齐全"，但句子显得过度用力、辞藻抢戏、作者存在感太强的场合。

## 目标

把文字从“作者在表演写作”拉回到“读者正在经历故事”。

## 一个总原则

- 风格可以有个性，但不该喧宾夺主。
- 当读者首先注意到的是作者在写得多漂亮，而不是人物正在承受什么，正文通常已经偏了。
- 默认优先追求：`干净、清晰、生动、真实`。

## 高风险信号

出现以下任两项，就要怀疑存在斧凿痕：

- 形容词、副词连续堆叠，像在给句子贴金箔。
- 同一段里连续出现抽象抒情、总结判断、作者式感慨。
- 本来一句能落地的动作，被改写成一整段诗化绕行。
- 场景推进很少，句子姿态很多。
- 比喻比动作更抢眼，读者记住了修辞，没记住发生了什么。
- 每段都像“我要写得不一样”，结果人物声音被作者声音盖住。

## 四步修整顺序

### 1. 先看这句是否真在服务故事

逐句问：

- 这句是在推进动作、补感知、显人物，还是只在炫句子？
- 删掉它，信息或气氛是否明显受损？
- 它是故事需要，还是作者想露一手？

若更接近后者，优先压缩或删除。

### 2. 把抽象美感改回可感知现场

优先转换：

- 抽象情绪 → 身体反应 / 呼吸 / 手部动作 / 视线停顿
- 宏大判断 → 具体物件 / 环境变化 / 关系动作
- 抒情句 → 当下可见后果

例：

- “空气里有一种命运般的沉重。”
  - 可改为：`门没关严，风一阵阵钻进来，吹得纸角一直拍桌。没人去按住。`

### 3. 控制修饰词密度

- 能删的形容词先删。
- 同一句里原则上不要让多个修饰词抢同一个名词。
- 若名词和动词已经够准，尽量不要再额外抬轿。
- 副词只在“少它就失真”时保留。

## 实操判断

- 同一自然段里，如果形容词和副词比动作动词更醒目，基本要减负。
- 若一个句子删掉修饰词后反而更有力，保留删后版本。

### 4. 让风格服从题材与场景压力

- 高压场景优先清晰、短促、可执行，不要句句雕花。
- 情绪余波场景可以稍微放宽节奏，但仍须有抓手，不能只剩氛围烟雾。
- 说明性段落要克制，宁可精准，不要华丽空转。

## 三个优先级

### 第一优先：读者先看懂

- 谁在做什么
- 为什么现在要做
- 这一步的代价或风险是什么

### 第二优先：读者能感到

- 人物在承受什么
- 现场在发生什么变化
- 局面是否在变紧、变坏、变复杂

### 第三优先：句子才谈风味

- 风味来自稳定、自然、贴合人物与题材的表达
- 不来自每段都强行造金句

## 允许保留风格的情形

以下场景可以保留较强风味，但仍不能失控：

- 明确承担人物视角过滤的贴脸段
- 章节结尾前的情绪余震
- 题材确实需要的审美表达位
- 反复出现、已经成为人物辨识的一种叙述习惯

前提是：

- 不遮蔽信息
- 不压扁动作
- 不冲掉钩子
- 不把所有角色写成同一种作者腔

## 快速自检

- 如果编辑或读者只看前三页，首先感到的是“文风好重”，而不是“故事开始了”，要继续减负。
- 如果朗读时总想在修辞上停顿，而不是顺着人物行动往下滑，说明仍有雕琢腔。
- 当读者不再明显注意你的写作风格、而是顺着故事往下走时，通常已经更接近专业区间。
