# 反常开场十机制速查

适用于章首、首段、宣传语式开场与第一屏抓眼句的快速构造。

## 使用前提

- 这些机制是为了更快抛出“第一钩”，不是为了制造空噱头。
- 选中任一机制后，都要尽快接回角色动作、即时风险与本章主卖点。
- 若只有怪句、没有后果，读者只会觉得作者在耍花活。

## 十种高频机制

### 1. 两难

- 角色眼下必须二选一，而且两边都要付代价。
- 适合写“保工作还是保尊严”“救一个人还是毁一条线”“说真话还是保关系”。
- 章首用法：第一段直接亮出选择题，第二段让角色被迫先做一个动作。

### 2. 矛盾

- 角色最该擅长的事，偏偏现在最做不到。
- 例如：老手当场失手、医生不敢进手术室、律师在关键时刻说不出话。
- 章首用法：先亮身份，再立刻让身份失灵。

### 3. 突兀

- 角色、物件或信息与当前环境明显不兼容。
- 例如：婚礼现场来了火化通知、幼儿园门口停着防暴车、白领会议桌上摆着止血钳。
- 章首用法：先给熟悉环境，再把“不该在这里的东西”推到读者眼前。

### 4. 悲剧

- 开头直接落在低点、残局或迟到一步的现场。
- 适合用来快速建立同情、焦虑、愧疚与补救压力。
- 章首用法：别先解释怎么走到这里，先让读者看见坏结果已经压过来了。

### 5. 借用

- 用大众熟悉的称号、角色、品牌气质给开头快速贴标签。
- 适合极短篇幅内建立形象，但必须克制，避免写成广告腔。
- 章首用法：只借一次，借完就要回到具体事件，不要连用多个比附。

### 6. 切身

- 把异常绑到读者熟悉的高频处境里，让人立刻感觉“这事可能也会落到我头上”。
- 例如：电梯、医院缴费口、家长群、绩效面谈、深夜回家路上。
- 章首用法：优先放在最熟悉的日常动作里起针，反常越近身越有效。

### 7. 奇观

- 用“唯一 / 第一次 / 从没见过 / 极端量级”制造第一眼冲击。
- 适合需要快速建立谈资、画面感或话题性的开头。
- 章首用法：奇观必须具体可见，最好不是抽象形容，而是能拍出来的东西。

### 8. 夸张

- 在不撒谎的前提下，把后果、压力或情绪推到更显眼的位置。
- 可用于“再迟一分钟就完了”“整层楼都知道了”这种可感的放大。
- 章首用法：夸张之后要马上给落点，证明这不是单纯喊大声。

### 9. 留白

- 先遮住最关键的结果、原因或身份，只给读者半扇门。
- 适合标题句、首句、段尾钩。
- 章首用法：遮住的必须是读者真的想知道的东西，而不是故作神秘的空白。

### 10. 揭秘

- 一上来就暗示“有件平常不会被看见的事，现在要被掀开了”。
- 适合秘密、潜规则、家丑、行业里话、关系翻面。
- 章首用法：别把答案一下讲完，先让读者知道“有人终于要说了”。

## 选型口诀

- 要立即压迫感：优先 `两难 / 悲剧 / 切身`
- 要身份反差感：优先 `矛盾 / 突兀 / 揭秘`
- 要传播感与单屏抓眼：优先 `奇观 / 夸张 / 留白 / 借用`

## 落地时的三条硬规则

1. 反常出现后，最多 1–2 段内就要接角色反应动作。
2. 反常必须与本章主卖点同向，不能只图吸睛。
3. 若读者看完首屏仍不知道“现在危险在哪儿”，说明机制选对了但落地失败。
