# 滑梯效应与首段安抚

适用于已经有标题、章名、首句方向，但首段还不够顺、不够钩、不够会接读者情绪的开头补强。

## 核心原则

- 标题的目的，是让人读第一句。
- 第一句的目的，是让人读第二句。
- 第二句的目的，是让人继续往下滑，而不是停下来怀疑作者在卖什么药。

## 先分析标题带来的情绪

读者被标题、章名或首屏异常吸引进来时，心里通常不是空白，而是已经带着一种情绪：

- 好奇：这到底怎么回事？
- 怀疑：这是真的吗？
- 愤怒：这事为什么会这样？
- 敬佩 / 惊讶：他怎么做到的？
- 焦虑：这会不会也落到我头上？

首段要做的第一件事，不是绕开这些情绪，而是**顺势接住并安抚**。

## 首段的四种高频补强动作

### 1. 回答标题里的第一层疑问

- 如果标题激起的是怀疑，首段尽快补一颗可信钉子：时间、现场、动作、身份、证据。
- 如果标题激起的是好奇，首段尽快给半个答案，再留更具体的问题。

### 2. 沿着标题关键词继续落地

- 标题里最有吸引力的那个词，首段不要丢。
- 要围绕它继续着墨，让读者思维自然衔接，而不是像换了一篇文章。

### 3. 尽快给读者一点值回票价

- 首段可以先给一个具体危险、一个明确困境、一个有用判断，或者一个能立刻代入的场景。
- 不要让读者点进来后，先吃三段热气，没有肉。

### 4. 用熟悉场景拉共鸣

- 可以是故事、痛点、权威、聊天式提问，但核心都是：让读者感觉“这事和我有关”。
- 若读者第一屏只觉得作者在自说自话，滑梯会断。

## 章首可直接套用的检查问句

- 标题最先激起的是哪种情绪？
- 第一段有没有顺着这股情绪往下接？
- 标题里最值钱的关键词，首段是否继续咬住了？
- 读者看完第一段后，是更想知道，还是已经想退出？

## 失败信号

- 标题很抓人，首段却像重新开会报背景。
- 标题抛了大钩子，首段完全不接，像换题。
- 首段只负责解释，不负责推进第二个问题。
- 标题让人觉得危险，首段却写得很安全。
