# 情境钩子与心理价值写法

适用于简介首句、短简介、引流版简介与需要在极短篇幅里立刻抓住读者的版本。

## 动笔前先问读者五问

简介再短，也最好先在脑中回答五个问题：

1. 这版简介最先写给谁看？
2. 这类读者眼下最痛的是哪一种坏处境？
3. 这本书能给他什么价值或改变？
4. 读完这段简介后，希望他先产生什么感受？
5. 这一版最核心的观点或承诺是什么？

若这五问答不清，简介很容易变成：设定很多、名字很多、气氛很多，但抓手没有。

## 坏情境 / 好情境压缩法

简介不是把世界观缩短，而是把“没有这本书时的坏情境”与“点开后会得到的好情境”压进极短篇幅。

### 先抓坏情境

- 主角眼下最受不了的“不什么”是什么：
  - 不安全
  - 不自由
  - 不被相信
  - 不被看见
  - 不够快
  - 不够体面
  - 不敢失手
- 首段最好能让读者立刻知道：主角现在卡在哪个坏处境里。

### 再给好情境承诺

- 简介后半段要让读者看到：如果主角继续往下走，他将摸到什么“什么感”。
- 常见可用感受：
  - 翻身感
  - 安全感
  - 自证感
  - 反击感
  - 被理解感
  - 更大真相带来的后劲

### 最短落地公式

- `主角 + 当前坏情境 + 反常入局点 + 更大承诺`
- `主角本来只想__，却因为__被拖进__；如果他不能__，就会__。`

## 简介首句的心理价值要求

- 首句不要只报设定，要尽量同时附加一种心理价值：危险、稀缺、同情、向往、反差或认同。
- 允许一句里同时完成“人 + 局 + 情绪”，不要求把设定一次讲完。
- 若只有 1–2 句空间，优先保住“主角现在最糟的处境”和“读者往下读会得到什么”。

## 四个常用首钩判断

简介首句若拿不准，可以自检它至少命中了以下四项中的一项，最好两项：

- **与我有关**：读者会不会觉得这类处境、身份、压力和自己有关
- **对我有用**：读者会不会感觉点开后能得到某种变化、方法或答案
- **挑动情绪**：是否让读者产生担忧、好奇、同情、向往或不甘
- **替我说话**：是否说中了读者心里那句不好概括、但一看就懂的话

若四项全空，通常说明首句只是“信息存在”，不是“钩子存在”。

## 借熟悉标签时的边界

- 可以借一个熟悉印象帮读者更快理解气质，但不能把简介写成广告语。
- 借标签后，必须立刻回到故事本身，不要连续比附多个标签。
- 若借完标签仍看不出主角处境，说明这句话只是好听，不算有效简介。
