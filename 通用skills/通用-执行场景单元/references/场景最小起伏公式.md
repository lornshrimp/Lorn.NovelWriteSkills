# 场景最小起伏公式

适用于把一个已有情节块压成真正有起伏、有位移、有回报的场景单元。

## 一、再用“常、境、变”拉出本场最小起伏

### 常：先让读者知道此刻原本应该怎样

- 不需要长篇背景，只需要让读者知道这地方、这关系、这流程“正常情况下”长什么样。
- 常态一立住，第一处反常才有力量。

### 境：把问题推进成困境

- 困境不是抽象地说“很难”，而是让角色真的卡住、慢一步、被盯住、失手、误判、来不及。
- 最好能同时具备：
  - 一项具体阻力
  - 一项马上可感的代价
  - 一次验证或补救动作

### 变：本场结束时必须有变化

- 变化可以是好，也可以是更坏。
- 最低要求：至少有一项状态发生位移：
  - 人物关系变了
  - 风险等级变了
  - 口径变了
  - 手里多了或少了一个关键东西
  - 对问题的理解变了

## 二、四个常见压缩公式

### 公式 A：日常裂缝型

`正常动作 → 轻微反常 → 角色复核 → 困境成立 → 留下新问题`

### 公式 B：职业破局型

`旁人误判 → 主角一句短判断 → 立刻动作 → 暂时破局 → 引出更贵后果`

### 公式 C：关系顶歪型

`表面平静 → 一句不对的话 → 位差暴露 → 对话升级 → 关系重排`

### 公式 D：追读供血型

`先给一点回报 → 马上抬高代价 → 再丢新变量 → 收在未完成动作上`

## 三、写完后用“具、情、简、意”快速复检

### 具体

- 少写“他很慌 / 很难受 / 很压抑”，多写动作、对白、物件和身体反应。

### 情感

- 本场最好只主打一条情绪主轴：恐惧、愤怒、羞耻、委屈、期待、心虚、庆幸等。

### 简单

- 能删掉却不影响本场目标的信息，就删。
- 术语、程序和设定，尽量压成读者一看就懂的生活化落点。

### 意外

- 至少有一处让读者觉得“原来问题在这里 / 原来这人会这么回 / 原来事情比表面更糟”。

## 四、失败信号

- 角色忙了半天，局面和开场时几乎没差。
- 只有氛围没有验证动作。
- 只有起，没有伏，没有中段回报。
- 场景结束时和开始时几乎一模一样。
