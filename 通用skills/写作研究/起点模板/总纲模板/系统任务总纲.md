# 系统任务总纲

## 1. 使用说明

### 1.1 大纲特点

- 可使用系统任务来驱动剧情发展，逻辑合理性要求低。
- 完成任务得到奖励的爽点直接有效，读者受众多。
- 写作门槛较低，适合加入到各种类型的网文中。
- 同类作品众多且系统只是辅助，需要设计好卖点才能出彩。

### 1.2 代表作品

- 会说话的肘子《大王饶命》

## 2. 系统设置

### 2.1 系统来源（主角视角）

- 穿越获得。
- 意外捡到。
- 天生自带。
- 人为安排。

### 2.2 沟通方式

- 系统单方面发布任务。
- 可进行机械式对话。
- 具有虚拟人格进行真实对话。
- 后台人工扮演进行真实对话。

### 2.3 奖励设置

- 完成任务的奖励最好和主角的需求绑定，示例：主角修炼需要加点就安排任务奖励点数。
- 奖励的物品等需要在未来剧情中用到。
- 奖励的物品等需要根据剧情发展进行更新。

### 2.4 系统解锁（非必要）

- 可随着主角成长解锁更多系统能力。

### 2.5 兑换商店（非必要）

- 可设置完成任务得到点数来增加兑换商店能力。
- 可设置高价格商品作为目标引导主角持续完成任务。
- 商品需要根据剧情发展进行更新。

### 2.6 面板属性（非必要）

- 可设置面板并过一段时间就提及面板来展示主角的实力变更（非必要）。
- 数值成长需要设计合理。
- 可建立其他角色的属性作为主角实力的参照系（非必要）。

### 2.7 属性示例

- 姓名。
- 年龄。

### 2.8 职业

- 等级/境界。
- 攻击类：战斗力/攻击力/武力值/智力值。
- 防御类：肉体强度/血量/防御力。
- 续航类：法力值/蓝量/体力。
- 资质类：天赋值/灵根等级/元素亲和力。
- 能力类：招式熟练度/技能等级。
- 其他常见属性：敏捷值/精神力/魅力值/抗性/灵魂强度/幸运值。

## 3. 任务设置

### 3.1 任务类型

- 剧情类：与其他角色进行对话/战斗等。
- 探索类：根据线索调查人物/地点/物品/历史等。
- 收集类：按照要求收集指定的材料等。
- 挑战类：限时要求完成指定的事件等。

### 3.2 触发条件

- 遇到危险。
- 新人物登场。
- 到达新场景。
- 到达新等级。
- 完成前置任务。
- 与其他角色沟通获得新情报。
- 时间周期性触发：每日/每月定时刷新等。
- 大事件发生：影响世界或者主角的命运等。

### 3.3 任务内容

- 任务名称。
- 任务类型：单人模式/团队模式。
- 任务背景。
- 具体任务内容。
- 任务难度。
- 任务奖励。
- 是否有时限性。
- 是否连环任务。

### 3.4 任务结算

- 任务完成度。
- 完成奖励。
- 未完成惩罚。

## 4. 剧情线

### 4.1 主线

- 主角不断变强。

### 4.2 支线（非必要）

- 获得系统。
- 系统升级。
- 解密系统的来源和目的。
