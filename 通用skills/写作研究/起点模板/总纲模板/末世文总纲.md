# 末世文总纲

## 1. 硬核向

- 求生是内在动机
- 绝望是整体基调
- 写实是具体场景

## 2. 轻松向

- 求生是内在动机
- 舒心是整体基调
- 乐子是调味剂

## 3. 7大元素

### 3.1 末世背景

- 丧尸/病毒
  - 人为
  - 自然变异形成
  - 天外而来
    - 携带病毒的陨石
    - 外星人传播
- 外星/怪物入侵
- 饥荒
- 永昼、永夜
- 天灾
- 极端环境
  - 极端温度（高、低温）
  - 酸雨、连续暴雨、毒气等
  - 空气污染
  - 陨石坠地球
- 植物泛滥

### 3.2 主角

- 先机
  - 重生
    - 前世被背叛、抛弃、陷害
    - 意外重生
    - 死亡重生
    - 获得部分先知先觉
    - 可选动机：弥补遗憾、改变未来走向
    - 可选动机：没有遗憾，只为生存
  - 非重生
    - 提前得到可靠消息
    - 没有信息，随机应变
- 队友
  - 父母
  - 闺蜜
  - 男、女友
  - 兄弟姐妹
  - 默契的朋友

### 3.3 资产配置（可以没有）

- 钱
  - 重生中彩票
  - 父母遗产
  - 大额贷款
  - 多年积蓄
- 安全屋
  - 学校宿舍
  - 出租屋
  - 单元楼
  - 别墅
  - 大平房

### 3.4 发展路线

- 囤物资
  - 依靠异能、系统
    - 直接或兑换获得
    - 完成任务获得
  - 依靠金钱提前购置
    - 需要一定资金，且拥有存储的地方（空间异能、房产）
    - 物资有限，后期依靠金手指或者其他途径补充
  - 依靠灾后搜寻
    - 没有准备，全靠在外面搜寻获得物资
    - 走这个流派，金手指是用来提升搜寻能力的，如强化身体、强化感知
  - 注：需配备存储物资的空间（系统空间、空间异能、储物格子/戒子）
- 强化自身
  - 完成系统任务获得奖励强化自身
  - 杀敌、搜寻特殊物品提升自身/异能
- 建造末日堡垒/基地
  - 系统/异能建造建筑
    - 完成任务或达成条件即可直接建造
    - 需要材料：收集材料、完成任务获得材料、兑换获得材料
  - 收拢幸存者
  - 可选
    - 金手指弄来人、机械、机器人帮忙建造发展

### 3.5 危机

- 抢夺物资，人性，遇到圣母，智斗恶人，面临包围绝境，物资紧缺

### 3.6 复仇

- 客观因素
- 间接仇怨
- 人为因素
  - 背叛、抛弃、陷害、伤害等
