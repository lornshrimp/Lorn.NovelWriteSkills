# 玄幻修行总纲

## 1. 故事核心

### 1.1 成长动因

- 前期是为了复仇，后来发现自己妹妹可能并没有死去，决定寻找妹妹，再到后面确认妹妹的灵魂印记确实散去了。

### 1.2 从第1章开始的底蕴

- 身负14境的龙韵、屠戮一界的数万亿生灵的煞气。

### 1.3 世间传说

- 逍遥天下有4位触及那个境界之人：1位剑仙、1位菩萨、1位道人、1位藏刀客。
- 剑仙斩龙妖，菩萨身化万千护人间万世周全，道人雷震群魔、刀客独守灵界大门承受千年孤寂（灵界是云七原先所在世界的一角，而这一角也包括其他洞天福地）。
- 穹荒天下有苍龙、玄武、白虎、朱雀。
- 因一场战事白虎身陨被魔族公主所救，苍龙在逍遥天下所斩，朱雀隐世不出，唯有玄武在位且辅佐那位（云七上一世所屠戮一界的煞气而成就的煞灵，此煞灵屠戮了白泽、害死了青丘狐妖、还宰了很多神兽）。
- 某座破碎的洞天住着1位上一界的余孽。
- 乱道天下有着1位斩杀过仙的凡人。

## 2. 修炼体系

### 2.1 开灵3小境

#### 2.1.1 解窍

- 六窍及之后就有了成为修士的可能。
- 七窍才是合窍的基本要求。
- 每个窍穴都能够承载一道灵韵。
- 灵韵分两种，一种是境界在十境之后死后产生，另一种是十境之下包括凡人，死后极端的执念形成的。

#### 2.1.2 启脉

- 将已有窍穴之间的体脉贯通，形成闭环。
- 脉络也有等级之分：凡、灵（下品、中品、上品）、圣、半仙、仙。

#### 2.1.3 凝纹

- 通过体脉循环的灵气开始反哺自身，到一定程度之后便能突破到炼气境。
- 体纹的凝练速度与体脉品质有关，至多3道气韵脉纹。

### 2.2 蜕凡4境

- 炼气
- 炼体
- 锻魂
- 合窍（应雷劫）
- 散气肉身求圣
- 斩血灵魂求神
- 三者契合

### 2.3 寻仙3境

#### 2.3.1 金丹

- 凝练出一颗与自身灵力同源的金丹。

#### 2.3.2 元婴

- 成就第2尊道外本身。

#### 2.3.3 寻韵

- 找到属于自己的气韵。

### 2.4 往圣3境

#### 2.4.1 琢天

- 打磨出道韵。

#### 2.4.2 通圣

- 道韵充斥着周身。

#### 2.4.3 飞升

- 可以拥有直接破开天下壁障的飞升资格。

### 2.5 仙2境

- 仙桥
- 筑型
- 凝实
- 含蕴
- 栩栩如生
- 掌道
- 将道法从天道手中剥夺或者与天道共用
- 空道

## 4. 杨洲城篇

- 背景
  - 自华云山的仙人招收门徒。
- 战力水平
  - 与云七同一辈的都在练气境以下。
  - 其他势力大多在八九境。
  - 老一辈的从十一境到十三境不等。
- 势力分布
  - 南北云家
  - 大立皇朝
  - 华云山
- 出场角色
  - 朱璨与轩辕琉儿
  - 本城圣人子女
  - 其他豪门纨绔子弟
- 剧情线
  - 为了能够加入宗门，变得更加强大。
  - 入仙门4关
    - 风雨图气运关
    - 震虎鼓气血关
    - 剑道山心智关（云七没有过）
    - 魁首争夺
  - 突破到了启脉。

- 剧情线

- 世界探索
  - 主线剧情的起因、经过、结局
