# 总纲模板目录说明

这个目录是一组偏 **起点风格网文** 的总纲模板库，适合在立项阶段、搭建世界观阶段、拆分剧情阶段和补结构阶段使用。

它不是一套只能“从头到尾照抄”的固定格式，而更像一个 **题材母板 + 结构工具 + 升级/地图工具 + 方法模板** 的组合仓库。实际使用时，通常不是只选一个文件，而是：

- 先选一个 **题材母板** 起盘；
- 再补一个或两个 **结构工具模板** 解决特定问题；
- 如果作品是升级流，再叠加 **升级体系模板**；
- 最后按自己的实际项目裁剪，不必全抄。

## 怎么用这批模板

一个比较稳的使用顺序是：

1. **先定题材与主承诺**
   - 例如恋爱、修仙、科幻、末世、都市高武、现实创业。
2. **再定结构方式**
   - 是多副本、明暗线、群像组织、地图切换，还是系统任务驱动。
3. **最后补增长逻辑**
   - 如果是升级流，再补修炼体系、换图升级、天才升级流等模板。

如果你现在的困难不是“不会写”，而是下面这些问题之一，可以优先这样选：

- **不知道选什么题材模板起盘**：看“题材母板类”
- **不知道怎么从灵感扩成大纲**：看 `拓展法大纲.md`、`九线写作总纲.md`
- **故事线太平、结构乱**：看 `简易明暗线总纲.md`、`副本单元剧总纲.md`
- **势力很多、关系容易乱**：看 `势力关系总纲.md`、`组织群像总纲.md`
- **升级节奏不好控**：看 `换图升级总纲.md`、`天才升级流总纲.md`
- **地图/副本切换容易割裂**：看 `莲花地图总纲.md`

- 先按**作品的主承诺**选主模板，而不是按你手上恰好哪份资料写得更细来选。
- 如果一个故事既有题材属性、又有很强的结构属性，优先判断**读者追更时最稳定的供血方式**是什么。
- 辅助模板只负责补结构，不替代主模板。
- 实在判断不出来时，主模板默认回退到 `九线写作总纲.md`。

| 只有灵感、还判断不清题材 / 结构 | `九线写作总纲.md` | `拓展法大纲.md`、`简略总纲.md` | 当你连主供血都还没钉住，就先用九线做主模板，别硬装自己已经想明白了。 |

如果你发现一个项目同时命中两三行，不要慌：

- 先问自己：**这本书如果删掉某个结构，读者还会不会觉得它本质上还是这本书？**
- 删掉后还成立的，通常是辅助模板；
- 删掉后主承诺就变了的，才应该是主模板。

## 模板分组总览

### 题材母板类

这类模板适合在“立项时”使用，帮你快速确定该题材常见的人设、冲突、节奏和读者预期。

| 文件 | 一句话定位 | 主要特点 | 适用场景 |
| --- | --- | --- | --- |
| `简略总纲.md` | 任意题材都能快速启动的极简母板 | 字段少、骨架清、适合先搭雏形 | 立项试错、脑暴期、想先快速出一个粗纲 |
| `凡人修仙总纲.md` | 凡人流修仙母板 | 强调“资质一般也能一步步变强”、讲求逻辑和危机感 | 凡人流、慢热修仙、成长压强要稳的作品 |
| `科幻通用总纲.md` | 科幻题材基础母板 | 重背景设定、科技系统、组织结构和主题表达 | 中短篇或长篇科幻、近未来/远未来设定搭建 |
| `恋爱文总纲.md` | 纯恋爱主导的基础母板 | 重男女主人设、阻碍因素、好感度推进 | 校园恋爱、都市恋爱、双主角关系驱动作品 |
| `恋爱炒股总纲.md` | 恋爱文的拆解演示母板 | 展示从灵感到细纲的完整落地过程，信息差和情绪推进清晰 | 日常恋爱、重生校园恋爱、想学习“怎么拆纲”的作者 |
| `诸天世界总纲.md` | 多世界穿梭母板 | 每个世界单独设定、单独推进，但要整体对标战力和收获 | 诸天流、无限流、跨世界成长型故事 |
| `末世文总纲.md` | 末世题材母板 | 区分硬核向/轻松向，梳理灾变类型、资源、基地、复仇线 | 丧尸、天灾、极寒极热、生存建设类作品 |
| `玄幻修行总纲.md` | 玄幻修行母板 | 自带较完整修炼层次与世界传说框架 | 传统玄幻、东方修行、世界观厚重型升级文 |
| `现实创业总纲.md` | 都市现实创业母板 | 事业/家庭/情感三线并行，现实阻力强 | 创业文、现实成长文、都市奋斗文 |
| `都市爱情成长总纲.md` | 都市爱情+成长母板 | 重主题、风格、成长与关系的双推进 | 都市情感、女性成长、关系修复/成长型爱情 |
| `都市高武总纲.md` | 都市+高武融合母板 | 战力体系和升级目标明确，强调危机拆解 | 都市高武、学院流、都市武道、机甲混合题材 |
| `重生纯爱总纲.md` | 重生恋爱母板 | 重前世/今生对照、人设塑造、情绪回补 | 重生恋爱、青春回溯、情感弥补型作品 |

### 方法/框架工具类

这类模板偏“怎么搭”，适合在不会拆纲、不会从点子扩成结构时使用。

| 文件 | 一句话定位 | 主要特点 | 适用场景 |
| --- | --- | --- | --- |
| `拓展法大纲.md` | 从一句话一路扩成卷纲的教学模板 | 从一句目标到一句角色、一句大纲、一段卷纲，步骤非常清楚 | 只有灵感，没有大纲；不会从点子扩结构 |
| `九线写作总纲.md` | 网文骨架检查模板 | 从主角、配角、技能、伙伴、装备、冒险等多个维度检查故事是否完整 | 快速补骨架、检查故事是不是“有东西可写” |
| `修炼体系总纲.md` | 通用修炼体系设计模板 | 从世界观、人物、能力体系、情节、章节规划一路往下搭 | 任何修炼/升级题材需要先搭体系时 |

### 结构工具类

这类模板不解决“写什么题材”，而解决“故事怎么组织”。

| 文件 | 一句话定位 | 主要特点 | 适用场景 |
| --- | --- | --- | --- |
| `简易明暗线总纲.md` | 双线叙事工具 | 把明线、暗线、交织节点拆开，适合做信息层次 | 悬疑、权谋、志怪、推理、公案、阴谋型剧情 |
| `副本单元剧总纲.md` | 单元剧/副本结构模板 | 每个副本都有背景、目标、障碍、人物、反转与回报 | 探案、除妖、怪谈、密室、闯关、仙侠副本 |
| `莲花地图总纲.md` | 多地图线索组织工具 | 以中心地图+外围地图组织主线、伏笔和回收 | 地图切换频繁、伏笔多、需要回环推进的长篇 |
| `势力关系总纲.md` | 势力网梳理工具 | 专门梳帮派、正道、外国势力、中立组织的结构 | 势力多、人物多、站队复杂的故事 |
| `组织群像总纲.md` | 群像组织模板 | 强调组织成员身份、加入目的、暴露风险与合作事件 | 组织流、会议流、秘密团体、马甲流群像作品 |
| `系统任务总纲.md` | 系统驱动剧情模板 | 从系统来源、任务类型、触发条件到结算都拆清楚 | 系统流、任务流、轻设定快节奏升级文 |

| `换图升级总纲.md` | 换地图升级模板 | 每张地图都有战力、资源、对手和升级回报，阶段感很强 | 传统升级流、打怪换图流、阶段 Boss 清晰的故事 |

## 每个模板更细一点的使用建议

下面这部分适合在你已经知道文件名，但不确定“该不该用它”的时候快速判断。

### `第四天灾总纲.md`

- **特点**：异世界设定、系统设定、玩家行为、阵营经营都放在同一个框架里。
- **适合**：主角不是单纯升级，而是“组织玩家干活、靠玩家推进世界变化”的故事。

### `动漫同人总纲.md`

- **特点**：强调在原作框架里外挂新组织、新能力、新人物关系。
- **适合**：需要既保留原作魅力，又让主角走出自己主线的动漫同人。

### `都市爱情成长总纲.md`

- **特点**：比纯恋爱模板更重“成长”与“主题表达”。
- **适合**：都市情感、成长修复、两性关系与人生选择并行的作品。

### `都市高武总纲.md`

- **特点**：重战力、重阶段目标、重敌人升级，节奏偏快。
- **适合**：都市学院、武道高武、超凡入世的升级故事。

### `凡人修仙总纲.md`

- **特点**：强调克制、谨慎、资源经营、危机感。
- **适合**：不想写“开局无敌”，而想写“普通人一点点苟上去”的修仙文。

### `副本单元剧总纲.md`

- **特点**：每个副本都有独立闭环，适合连载里反复制造新鲜感。
- **适合**：怪谈、探案、秘境、试炼、都市异闻等“一个单元一个钩子”的写法。

### `换图升级总纲.md`

- **特点**：地图递进、反派递进、资源递进非常清楚。
- **适合**：传统升级爽文，尤其适合“每一段写完就换新舞台”的长篇。

### `简略总纲.md`

- **特点**：信息最少、启动最快。
- **适合**：先试题材、先试人物、先找故事感觉，不适合直接拿来做终版大纲。

### `简易明暗线总纲.md`

- **特点**：适合把表面剧情和底层真相拆开。
- **适合**：悬疑、公案、权斗、暗黑修仙、带阴谋层的故事。

### `九线写作总纲.md`

- **特点**：像一个“网文骨架自检表”。
- **适合**：你已经有故事想法，但不确定是不是少了伙伴、技能、势力、身世这类供血项。

### `科幻通用总纲.md`

- **特点**：更偏故事架构与科幻背景设定，不是硬核科学论文式模板。
- **适合**：中短篇科幻、软科幻、未来社会类科幻作品。

### `莲花地图总纲.md`

- **特点**：强调“地图切换不散，线索回收不断”。
- **适合**：长篇冒险、诸界穿梭、地图机关/秘境/远行型作品。

### `恋爱炒股总纲.md`

- **特点**：不是纯模板，更像“拆解示例 + 演示文稿”。
- **适合**：想学会怎么从一个恋爱灵感拆成细纲的人。

### `恋爱文总纲.md`

- **特点**：关系推进最直接，重点在“为什么在一起、为什么还没在一起”。
- **适合**：纯恋爱、校园恋爱、轻松日常恋爱。

### `末世文总纲.md`

- **特点**：灾变类型、资源逻辑、先机类型、基地建设都考虑到了。
- **适合**：末世囤货、天灾生存、基地发展、复仇向末世文。

### `势力关系总纲.md`

- **特点**：把组织、派系、势力面拆得很细，适合单独拿来补档。
- **适合**：主线已经有了，但你发现势力体系还很虚的时候。

### `拓展法大纲.md`

- **特点**：最适合新手或者临时卡壳时救场。
- **适合**：只有一句话设想，完全不会往下扩的时候。

### `天才升级流总纲.md`

- **特点**：比换图升级更强调“主角是天才”带来的爽感与效率。
- **适合**：主角高天赋、多职业、开挂合理化比较强的故事。

### `系统任务总纲.md`

- **特点**：把系统流最容易写乱的任务逻辑整理好了。
- **适合**：系统奖励、任务触发、成长反馈是核心驱动力的作品。

### `现实创业总纲.md`

- **特点**：非常重现实压力与多线交织，不适合纯爽文式空想创业。
- **适合**：现实都市、创业、家庭矛盾、情感成长交错的作品。

### `修炼体系总纲.md`

- **特点**：更像设计模板而不是题材模板。
- **适合**：你先要搭体系，后面再决定写玄幻、仙侠还是高武。

### `玄幻修行总纲.md`

- **特点**：世界传说和修炼层次都给得更厚重、更偏东方玄幻。
- **适合**：想写一个体系完整、设定感强的修行故事。

### `重生纯爱总纲.md`

- **特点**：重情绪回补、人物前世遗憾与今生反差。
- **适合**：青春重生、恋爱回补、遗憾弥补类故事。

### `诸天世界总纲.md`

- **特点**：重点是“每个世界都能独立成段，但还服务整体成长”。
- **适合**：诸天流、无限流、快穿式长篇，但偏网文化叙事。

### `组织群像总纲.md`

- **特点**：擅长处理多成员、多身份、多合作事件。
- **适合**：群像组织、秘密结社、会议流、马甲身份互相试探的故事。

## 容易混淆的模板区别

### `简略总纲.md` vs `拓展法大纲.md`

- `简略总纲.md` 是 **成品框架**；
- `拓展法大纲.md` 是 **从灵感往成品走的方法**。

### `修炼体系总纲.md` vs `玄幻修行总纲.md`

- `修炼体系总纲.md` 偏 **通用设计工具**；
- `玄幻修行总纲.md` 偏 **具体玄幻母板**。

### `换图升级总纲.md` vs `天才升级流总纲.md`

- `换图升级总纲.md` 偏 **阶段/地图递进**；
- `天才升级流总纲.md` 偏 **主角天赋与成长爽感**。

### `副本单元剧总纲.md` vs `莲花地图总纲.md`

- `副本单元剧总纲.md` 偏 **一个单元一个闭环**；
- `莲花地图总纲.md` 偏 **多个地图互相供血、伏笔回收**。

### `系统任务总纲.md` vs `第四天灾总纲.md`

- `系统任务总纲.md` 解决的是 **主角与系统** 的关系；
- `第四天灾总纲.md` 解决的是 **主角、系统、玩家群体、异世界** 的多方关系。

## 推荐搭配方式

如果你想更快起盘，可以直接按下面的组合来：

- **凡人修仙**：`凡人修仙总纲.md` + `势力关系总纲.md` + `莲花地图总纲.md`
- **玄幻升级**：`玄幻修行总纲.md` + `换图升级总纲.md` + `天才升级流总纲.md`
- **校园恋爱/重生恋爱**：`恋爱文总纲.md` + `恋爱炒股总纲.md` + `重生纯爱总纲.md`
- **系统流**：`系统任务总纲.md` + `都市高武总纲.md` / `第四天灾总纲.md`
- **单元悬疑/志怪**：`副本单元剧总纲.md` + `简易明暗线总纲.md` + `组织群像总纲.md`
- **都市现实成长**：`现实创业总纲.md` + `都市爱情成长总纲.md`

## 最后一句建议

这批模板最好的用法，不是“只挑一个照着填”，而是：

- **题材母板** 决定故事承诺；
- **结构工具** 解决叙事问题；
- **升级/地图模板** 解决连载供血；
- **方法模板** 帮你从灵感走到可执行大纲。

把它们当成积木，会比把它们当成表格更好用。
