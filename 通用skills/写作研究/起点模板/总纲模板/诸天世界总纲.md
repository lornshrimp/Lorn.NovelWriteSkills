# 诸天世界总纲

## 1. 使用说明

### 1.1 大纲特点

- 以世界作为独立单元，一个世界里面包含一个完整故事。
- 每个世界需要有详细的世界观设定，也可以借鉴已有的成熟世界。
- 需要注意每个世界开展的故事性尽量要结合世界的特点。

### 1.2 代表作品

- 裴屠狗《诸天投影》

## 2. 故事核心

### 2.1 成长动因

- 主角是为了什么而不断变强。

### 2.2 金手指

- 主角独有的能力或者特质。

### 2.3 穿梭方式

- 例如：灵魂转移。

## 3. 世界1

- 世界背景
  - 描述世界的基本历史和风土人情。
- 战力水平
  - 世界的普通战力和顶级战力在整个作品战力体系中的对标水平。
- 势力分布
  - 登场的主要势力和敌对关系。
- 修炼体系
  - 修炼各层级的名称和特点。
- 出场角色
  - 登场的主要角色。
- 剧情线
  - 前往此世界的目的。
- 世界探索
  - 主线剧情的起因、经过、结局。
- 主角收获
