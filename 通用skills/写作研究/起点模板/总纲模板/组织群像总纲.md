# 组织群像总纲

## 1. 使用说明

### 1.1 大纲特点

- 通过组织的形式可以让主角在不同阶段交易到合适的资源
- 与组织内其他成员的互动可以很好的成为主线或者支线的线索
- 需要持续对于组织成员进行画像处理来加深印象

### 1.2 代表作品

- 爱潜水的乌贼《诡秘之主》
