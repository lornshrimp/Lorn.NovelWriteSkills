# 换图升级总纲

## 1. 使用说明

### 1.1 大纲特点

- 剧情脉络明显，主角只需要不断地在各个地图中打怪升级到顶点再换新地图即可。
- 不同地图中的战力体系一脉相承，共用一套世界观。
- 需要对不同地图的战力进行统一设计，避免后期战力膨胀。
- 换地图存在割裂感，剧情容易审美疲劳。

### 1.2 代表作品

- 我吃西红柿《雪鹰领主》

## 2. 故事核心

### 2.1 成长动因

- 主角是为了什么而不断变强（升级）。

### 2.2 金手指

- 主角独有的能力或者特质。

### 2.3 升级攻略

- 主角的升级思路。

### 2.4 爽点

- 公认的修行天才。
- 战斗力强，可越级击杀。
- 拥有各种顶级装备。

## 3. 升级体系

### 3.1 职业类型

- 职业1的名称和特点。
- 职业2的名称和特点。
- 职业3的名称和特点。

### 3.2 修炼层次

- 层次1的名称、效果（伤害、寿命等）和升级需要的条件（各类资源、个人天赋等）。
- 层次2的名称、效果（伤害、寿命等）和升级需要的条件（各类资源、个人天赋等）。
- 层次3的名称、效果（伤害、寿命等）和升级需要的条件（各类资源、个人天赋等）。
- 层次4的名称、效果（伤害、寿命等）和升级需要的条件（各类资源、个人天赋等）。
- 层次5的名称、效果（伤害、寿命等）和升级需要的条件（各类资源、个人天赋等）。

### 3.3 内部等级

- 单一层次里面的等级划分，如初期、中期、后期、圆满。

### 3.4 地图1

- 战力分布
  - 地图中的战力范围是从哪到哪。
  - 地图的高端战力的数量。
- 势力分布
  - 登场的主要势力和敌对关系。
- 地图资源
  - 修炼需要的资源在哪里有。
- 升级办法
  - 举例：加入门派获取资源。
- 对手1（小反派）
  - 对手信息（身份、势力、性格）。
  - 结仇原因（利益、情感）。
  - 事件过程（起因、发展、高潮、结局、引出新人物或新事件）。
  - 击败收获（资源、声望）。
- 对手2（大反派）
  - 结仇原因（利益、情感）。
  - 事件过程（起因、发展、高潮、结局、引出新人物或新事件）。
  - 击败收获（资源、声望）。
  - 对手信息（身份、势力、性格）。
- 对手3（终极反派）
  - 结仇原因（利益、情感）。
  - 事件过程（起因、发展、高潮、结局、引出新人物或新事件）。
  - 击败收获（资源、声望）。
  - 对手信息（身份、势力、性格）。
  - 配套升级
    - 功法升级（达到新层级或换新功法）。
    - 武器升级（锻炼新属性或换新武器）。
    - 防具升级（锻炼新属性或换新防具）。
    - 宝物升级（锻炼新属性或换新宝物）。
    - 排名升级（榜单的名次提升）。
    - 特有能力升级（天赋、职业技能等）。
