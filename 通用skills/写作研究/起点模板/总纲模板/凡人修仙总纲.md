# 凡人修仙总纲

## 1. 使用说明

### 1.1 大纲特点

- 核心是要表现出主角在资质/资源一般的情况通过自己的努力一步步变强（ 平凡人成就不凡事）
- 世界观通用，修炼体系、辅助职业、宝物道具等设定成熟，读者接受度高
- 剧情逻辑要求严谨，出场角色的为人处事要真实可信
- 前期慢热时要避免灌水，增加小情节不断节奏，中后期避免升级快，保障危机感

### 1.2 代表作品

- 忘语《凡人修仙传》

## 2. 金手指（非必要）

- 常见为辅助类型，有限制条件，不能过于强大

### 2.1 常见类型

- 战斗天赋类：危险感知、无限蓝条、抗揍等
- 辅助天赋类：寻宝、望气、、好运、辅助职业天赋异禀等
- 道具类：加快修炼、提高丹药品质、改进功法等

## 3. 人设要求（仅参考）

- 修仙资质一般或较差
- 有一项或者多项技艺（辅助职业）来获取资源
- 智商在线，决策前考虑多个方面（考虑不全是正常情况）
- 性格平和，不极端不圣母
- 做事果断，该出手时就出手，打不过就跑

## 4. 情节设计

- 围绕着主角量力而行来设计剧情：风险小就干，风险中就避，风险大就跑。

### 4.1 开局类型

- 出身散修，偶得奇缘获得成长机会
- 出身武侠，修炼到世俗顶点获得仙缘
- 出身宗门弟子，尔虞我诈中逐步上位
- 出身破败修仙家族，解决家族困境重现辉煌

### 4.2 常见剧情

- 赚取灵石
- 学习法术
- 寻求灵药
- 路遇斗法
- 拜入宗门
- 前辈打压
- 宗门大会
- 黑市交易
- 拍卖风波
- 秘境夺宝
- 高阶追杀
