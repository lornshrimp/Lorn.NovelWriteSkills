# 榜单检索与样本池确定规则

本规则用于把“找同题材最优秀作品”从随手点开几本热门书，收束成**口径明确、可复盘、可解释**的样本池建设动作。

## 一、默认参数

- 默认目标平台：`起点中文网`
- 默认榜单规模：`Top10`
- 默认目标：锁定与当前项目**商业竞争位最接近**的强样本，而不是广义意义上的“所有出名作品”

## 二、榜单口径选择顺序

### 2.1 若用户未指定榜单类型

默认按以下顺序判断：

1. **与当前项目最接近的商业竞争位榜单**
   - 例如：畅销、热销、月票、分类强榜等
2. **若当前项目明显处于新书策略阶段**
   - 可优先补看新书榜 / 新锐榜 / 近期强势样本
3. **若题材边界模糊**
   - 允许组合两个相近榜单建立样本池，但必须写清主榜与辅榜的职责

### 2.2 若用户明确指定榜单类型

- 直接按用户指定执行；
- 但若指定榜单与当前项目阶段明显不匹配，必须在报告中显眼写出：这会带来什么偏差。

## 三、搜索与核验要求（强制）

- 必须使用多种可用搜索方式做交叉检索，不得只依赖单一搜索源。
- 至少完成：
  1. **一次主检索**：锁定榜单候选；
  2. **一次交叉检索**：确认榜单与题材标签是否一致；
  3. **一次结果核验**：确认样本没有明显跑错赛道。
- 若官方榜单页难以直接打开，允许通过多组关键词、不同搜索源和多篇外部页面还原样本池；但必须把盲区写清楚。
- 对起点中文网样本，可直接使用 `https://www.qidiantu.com/` 作为补充检索与核验入口：榜单页可回看推荐位，首订页可看新书首订与真实粉丝增量，徽章页可看荣誉徽章口径，盟主页可看盟主数量，作品详情 / 数据页可补充推荐票、收藏数、真实粉丝数、月票数、字数等；但至少仍要配一条官方页或其他外部来源做交叉核验。
- 若官方作品页、目录页或试读页因验证码、反爬、登录门槛或频控受阻，允许转向**更稳定、可核验的替代来源**补足样本信息，例如官方 App 试读、作者或平台公开物料、媒体报道、读者书评、论坛讨论、剧情梳理、百科 / 设定页等；**不要把来源不明、不可稳定核验的全文镜像页或转载聚合页列为核心检索来源。**

## 四、推荐搜索词模板

以下搜索词可按需拼接：

- `起点中文网 {题材} 榜单`
- `起点 {题材} Top10`
- `起点 {题材} 月票榜`
- `起点 {题材} 畅销榜`
- `起点 {题材} 热门作品`
- `{题材} 起点 推荐作品`
- `{作品名} 起点图`
- `site:qidiantu.com {作品名}`
- `{作品名} 起点`
- `{作品名} 目录`
- `{作品名} 书评`
- `{作品名} 评论`
- `{作品名} 读后感`
- `{作品名} 拆解`

> 若题材边界不止一个叫法，必须并行检索多个同义口径，例如“都市悬疑 / 都市惊悚 / 都市怪谈 / 都市异闻”等，并在最终样本池说明里解释取舍。

## 五、样本入池规则

只有满足以下至少多数条件的作品，才应进入正式 TopN 样本池：

- 与当前项目属于**同平台或高度相近平台语境**；
- 题材承诺、读者预期、商业竞争位大体一致；
- 能找到足够多的外部证据支持后续深拆；
- 不是只靠作者名气被讨论，而是作品本身确有可分析价值；
- 不是明显跑偏到另一赛道的“伪同题材样本”。

## 六、样本排除规则

出现以下情况时，默认不应直接纳入主样本池：

- 平台机制明显不同，导致读者预期完全变形；
- 表面同题材，实则主卖点完全不同；
- 几乎找不到可用外部证据，只能靠模糊印象猜测；
- 作品讨论度很高，但核心优势明显依赖影视改编、作者存量或跨圈热度，而不是文本本体；
- 与当前项目赛道差距过大，把它纳入反而会误导分析维度。

> 若某部作品知名度极高、但赛道并不完全一致，可以放入“旁证样本”或“边界样本”，但不能伪装成主样本。

## 七、榜单与样本池记录格式（建议）

| 字段 | 说明 |
| --- | --- |
| 榜单名称 | 最终采用的榜单口径 |
| 目标平台 | 起点中文网 / 其他 |
| 题材口径 | 例如：都市悬疑 / 异能志怪 |
| TopN 数量 | 如 10 |
| 检索时间 | YYYY-MM-DD |
| 主要搜索词 | 本次主要搜索词组合 |
| 主要搜索来源 | 榜单页、作品页、书单页、推荐文等 |
| 交叉核验来源 | 至少 1 类 |
| 主样本名单 | 最终纳入的作品清单 |
| 排除样本与原因 | 为什么没有纳入 |
| 当前盲区 | 还原样本池时仍存在的不确定点 |

## 八、样本池确定后的最低动作

样本池一旦锁定，接下来至少要做：

1. 为每本作品建立证据卡；
2. 对每本作品至少补齐“目录 / 正文片段 / 评论材料”三类证据；
3. 再决定哪些作品是“全维度强标杆”，哪些只是“局部强样本”；
4. 最终为每本作品单独落盘详细报告，而不是把全部样本挤成一份混合摘要。

## 九、常见误区

- 只因为作品很火，就自动视为最适合当前项目对标的样本。
- 只选自己熟悉或喜欢的书，而不是按竞争位选样本。
- 用不同榜单、不同时间、不同赛道的书拼凑一个看上去很强的名单，却不解释混样口径。
- 没有完成样本池核验就开始逐本下判断。
- 用“大家都说好看”代替“它具体为什么强”。
- 因官方源受阻，就默认转去来源不明、不可稳定核验的全文镜像页硬抓目录、正文与章节清单。
