# 通用-输出B站版｜跨题材共性与迁移来源

本文件记录 `通用-输出B站版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `异能志怪/.github/prompts/输出B站版.prompt.md`
- `都市悬疑/写作研究/B站平台长篇中等节奏都市悬疑小说文笔风格与代入感写作指南.md`

## 当前可稳定上收的 B站 平台共性

- 强调年轻读者可读性、讨论度、视听化场景感与节奏清晰度。
- 输出结构固定为“正文 + ## 作者有话说”。
- 标题与开头都应更适合年轻读者快速进入语境，但仍必须保持小说体裁。
- 可适度放大可讨论线索、规则漏洞、反差点与“弹幕化情绪节点”，但这些互动点必须长在剧情里。
- 允许轻吐槽、轻梗感与节奏切分，但不得写成视频解说稿、书单口播稿或剧情复盘帖。
- 平台风格必须先于降相似度，不得先写泛中文平台稿再套 B站 壳。

## 当前仍保留在题材层的内容

- 题材本身的规则链、现实锚点、人物代价与危险空间。
- 题材专属的讨论感重心、异常显影方式与执行补丁。

## 使用边界

- 命中 `通用-输出B站版` 时，必须同时加载对应题材的 `题材名-输出B站版`。
- 迁移期内若仍需回溯细节，以题材 refs、平台研究文件与源 Prompt 为补充。
