# 通用-输出小红书版｜跨题材共性与迁移来源

本文件记录 `通用-输出小红书版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `异能志怪/.github/prompts/输出小红书版.prompt.md`
- `都市悬疑/写作研究/小红书平台 200 万字以上中等节奏都市悬疑小说受欢迎文笔风格与代入感构建研究.md`

## 当前可稳定上收的小红书平台共性

- 强调体感、画面感、即时情绪反馈、凡人共情与分享冲动。
- 输出结构固定为“正文 + ## 作者有话说”。
- 标题与开头都应更适合移动端快速吸附，优先让“日常场景 + 诡异细节 + 主角情绪”尽快显形。
- 第一人称或近距离限制性视角、口语化表达、日常感官细节与轻松弛调节，是小红书平台更稳定的共性抓手。
- 小红书传播常依赖“短种草 + 长阅读”，但当前输出对象仍必须是小说正文，不得写成笔记、攻略或推文。
- 平台风格必须先于降相似度，不得先写泛中文平台稿再套小红书壳。

## 当前仍保留在题材层的内容

- 题材本身的规则链、现实锚点、危险感与人物代价。
- 题材专属的都市空间压迫、职业线索与执行补丁。

## 使用边界

- 命中 `通用-输出小红书版` 时，必须同时加载对应题材的 `题材名-输出小红书版`。
- 迁移期内若仍需回溯细节，以题材 refs、平台研究文件与源 Prompt 为补充。
