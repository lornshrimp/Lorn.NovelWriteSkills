# 分部与分卷审阅报告增补

## 报告前置要求

- 先写清：上一级分配给本层的内容，哪些已落地，哪些缺失，哪些只是挂名未落实。

## 报告里必须写重的部分

### 1. 可执行性与可拆章性

- 作者是否能直接据此拆章
- 哪些段落仍是评论大纲，不是执行大纲

### 2. 节奏与钩子策略

- 小高潮 / 大高潮的位置
- 钩子类型是否轮换
- 哪几章最容易“章首猛但中段塌”
- 建议补什么类型的中段回报

### 3. 前五章留存、卷中回报与续购诊断

- 前 5 章是否形成连续追读链
- 卷中最容易失温的章组在哪里，为什么
- 单元案是否既能独立成立，又能反哺主线
- 卷末是否留下更贵的新缺口，能推着读者进入下一卷

### 4. 平台节拍与拆章适配

- 当前卷更偏哪类平台成品，为什么
- 卷长 / 章长 / 钩子密度与目标平台是否匹配
- 是否存在“结构能看，但一拆章就失温”的风险

### 5. 具体修改建议

建议用“插入位置 + 章组补丁 + 预期收益”的格式，例如：

- 在【某章组】增加【中段回报】，避免本段空转。
- 将【当前钩子】改为【转折 / 揭露 / 危机】，避免连续同型疲劳。
- 在【本部 / 本卷承接清单】补列【缺失的上游分配项】，并落到【对应章组 / 工件】。
- 在【卷末】补一个【更贵的新缺口 / 跨卷后果】，提升下一卷续购驱动力。
- 在【前 5 章】补一条【回应旧问题 + 新增更危险问题】的追读链，避免只靠首章独木支撑。
