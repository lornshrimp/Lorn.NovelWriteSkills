# 投稿材料包清单与排序

## 核心原则

投稿材料包的排序不是为了作者好整理，而是为了让编辑按最省力的顺序完成判断：

1. 这是什么书；
2. 为什么值得继续看；
3. 这书会不会兑现承诺；
4. 作者有没有能力继续写下去。

## 常见必备件（正式投稿包）

1. 书名 / 目标投放名
2. 一句话卖点
3. 内投版简介
4. 内投版正文（连续样章）
5. 内投版大纲
6. 组包自检

## 建议排序（起点 / 连载平台默认口径）

1. **书名 + 一句话卖点**：先让编辑知道这是什么书。
2. **内投版简介**：再让编辑判断有没有点击与继续看的理由。
3. **内投版正文**：再证明前门真的抓得住人。
4. **内投版大纲**：再证明这不是只会开头的稿子。
5. **组包自检**：最后交代当前最强、最弱与残留风险。

## 每件材料各自解决什么问题

- **书名 / 一句话卖点**：解决“这是什么题材、卖点清不清”。
- **简介**：解决“我为什么愿意点开、继续看”。
- **正文**：解决“这套承诺是不是真的落进了开篇”。
- **大纲**：解决“它能不能支撑首卷和长线，不是一锤子买卖”。
- **组包自检**：解决“作者有没有清楚认识当前这包材料的真实强弱”。

## 可选增强件

- 平台适配说明
- 爽点 / 悬念 / 单元案矩阵摘要
- 作者写作节奏与更新能力说明
- 主角 / 人物补充卡
- 规则 / 设定补充卡
- 已有平台评估结论摘要

## 不推荐排序

- 先扔长大纲，再补简介与样章
- 先放作者说明，再放可读材料
- 先放设定百科，再放正文
- 把人物、设定、总纲全堆前面，让编辑迟迟看不到前门
